Для распознавания лиц их нужно собрать, для этого используем модели haarcascade_frontalface_default и haarcascade_profileface для выделения лиц на видео и сохранения их в коллекцию. Я собирал примерно по 150-250 вариантов для каждого человека. Скрипт collect_images.py расположен в репозитории, ссылка на который будет в конце статьи. В video = cv2.VideoCapture(0) указываем источник:
•	0 – Веб камера
•	Можно использовать RTSP поток
•	Можно использовать видеофайл
